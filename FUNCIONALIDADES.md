# Génesis Electric - Funcionalidades del Proyecto

Este documento describe las principales páginas y características de la aplicación web de Génesis Electric.

## Páginas Principales

*   **/ (Inicio):** Página de inicio con un carrusel de imágenes (Hero Slider) y secciones de marketing.
*   **/products:** Página de listado de productos con filtros y búsqueda.
*   **/products/:id:** Página de detalle de un producto específico.
*   **/cart:** Carrito de compras donde los usuarios pueden ver y modificar los productos que han agregado.
*   **/checkout:** Página de pago donde los usuarios finalizan su compra.
*   **/favorites:** Lista de productos favoritos guardados por el usuario.
*   **/login, /register, /signup:** Páginas para el inicio de sesión y registro de usuarios.
*   **/forgot-password, /reset-password/:token:** Páginas para el proceso de recuperación de contraseña.
*   **/profile:** Perfil de usuario con sub-secciones para ver información de la cuenta, órdenes, direcciones y configuración de seguridad.
*   **/service-order:** Formulario para solicitar una orden de servicio técnico.
*   **/btu-calculator:** Herramienta para calcular los BTUs necesarios para un aire acondicionado.
*   **/privacy-policy, /terms-and-conditions:** Páginas con información legal.

## Características Principales

*   **Autenticación de Usuarios:** Sistema completo de registro, inicio de sesión, y recuperación de contraseña. Los usuarios tienen perfiles donde pueden gestionar su información personal.
*   **Catálogo de Productos:** Los usuarios pueden navegar por los productos, ver detalles, y añadirlos al carrito o a su lista de favoritos.
*   **Búsqueda y Filtros:** Funcionalidad de búsqueda de productos y filtros por diferentes criterios.
*   **Carrito de Compras:** Carrito persistente que guarda los productos del usuario.
*   **Proceso de Pago (Checkout):** Integración con la pasarela de pagos Wompi para procesar las transacciones de forma segura.
*   **Gestión de Perfil de Usuario:** Los usuarios pueden ver su historial de órdenes, gestionar sus direcciones de envío y actualizar su información de perfil y seguridad.
*   **Panel de Administración:**
    *   Gestión de productos (crear, editar, eliminar).
    *   Gestión de órdenes de servicio.
    *   Gestión de clientes.
    *   Configuración del Hero Slider de la página de inicio.
*   **Calculadora de BTU:** Herramienta útil para que los clientes elijan el aire acondicionado adecuado.
*   **Diseño Responsivo:** La aplicación está diseñada para funcionar en diferentes tamaños de pantalla.
